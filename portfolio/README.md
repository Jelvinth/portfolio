# Jelvinth Lakshmanan Portfolio

A professional, fully responsive web portfolio built with HTML, CSS, and Vanilla JS.
Features dynamic 3D interactions using `vanilla-tilt.js` and modern glassmorphism design.

## Structure
- `/assets/css/` - Custom CSS and animations
- `/assets/js/` - Interactive scripts
- `/assets/images/` - Site images and profile pictures
- `/assets/docs/` - Downloadable assets (Resume/CV)

## Tech Stack
- HTML5
- Vanilla CSS3 (Variables, Flexbox, Grid)
- Vanilla JavaScript
- Feather Icons
- Vanilla-Tilt.js for 3D UI

## Running Locally
You can run this project locally using Python's built-in HTTP server or any live server extension:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
